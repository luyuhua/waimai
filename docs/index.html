🚀 步骤 3：在任意网页使用

打开任意网页（如美团商家版），点击书签栏中的「提取页面结构」，然后：

  • 按 F12 打开开发者工具
  • 切换到 Console 标签查看输出结果
  • 页面会高亮显示所有可交互元素
💡 提示：
  • 这是书签工具，不是网页链接，必须拖拽到书签栏
  • 拖拽后，在任意网页点击书签即可执行脚本
  • 部分网站可能有 CSP 安全策略，脚本无法注入

💾 数据已保存到全局变量:
  - window.__domResult (完整结果)
  - window.__interactiveElements (可交互元素)
💡 操作示例： 
// 点击第 0 个元素
__interactiveElements[0].ref.click();

// 查找包含"出餐"文本的元素
findElementByText(__domResult.map, '出餐').ref.click();

// 清除页面高亮
clearHighlights();
